## 系统概述

机房动环监测系统 (DCIM) 是一套专业的数据中心基础设施管理系统，提供以下核心功能：

- **环境监控**: 温湿度、电力、安防等实时监控
- **设备管理**: 设备信息管理、状态监控
- **告警管理**: 多级告警、实时推送、声音提醒
- **历史数据**: 数据查询、趋势分析、报表导出
- **用电管理**: PUE监控、能耗统计、节能建议
- **系统设置**: 阈值配置、用户管理、系统日志

### 访问地址
- 系统地址: http://localhost:3001
- API文档: http://localhost:8000/docs

## 实时监控仪表盘

仪表盘是系统的首页，提供机房整体状态的实时概览。

### 功能区域

#### 1. 统计卡片
显示关键指标统计：
- **监控点位**: 总点位数量
- **正常点位**: 运行正常的点位数
- **告警点位**: 当前告警的点位数
- **离线点位**: 失联的点位数

#### 2. 实时数据表格
显示所有点位的实时数据：
- **点位编码**: 唯一标识符 (如 A1_TH_AI_001)
- **点位名称**: 易读名称 (如 A1区温度)
- **类型**: AI/DI/AO/DO 四种类型
- **当前值**: 实时数据 + 单位
- **状态**: 正常/告警/离线
- **更新时间**: 最后更新时间

#### 3. 最新告警
显示最近的活动告警：
- 告警级别 (紧急/重要/次要/提示)
- 告警点位和信息
- 告警时间

### 数据刷新
- 系统每 10 秒自动刷新一次
- 可点击"刷新"按钮手动刷新

## 点位管理

点位是监控系统的基本单元，代表一个监控或控制对象。

### 点位类型
| 类型 | 说明 | 示例 |
|------|------|------|
| AI | 模拟量输入 | 温度、湿度、电压、电流 |
| DI | 开关量输入 | 门禁状态、设备运行状态 |
| AO | 模拟量输出 | 温度设定值、风速调节 |
| DO | 开关量输出 | 设备启停控制 |

### 查看点位列表
1. 点击左侧菜单"点位管理"
2. 可按类型、区域、状态筛选
3. 支持搜索点位名称或编码

### 点位详情
点击点位可查看详细信息：
- 基本信息: 编码、名称、类型、单位
- 数据范围: 最小值、最大值、精度
- 采集配置: 采集间隔、存储间隔
- 实时状态: 当前值、状态、更新时间

### 点位编码规则
格式: `{区域}_{设备类型}_{点位类型}_{序号}`

示例:
- `A1_TH_AI_001`: A1区温湿度传感器AI类型第1个点位
- `A1_UPS_DI_001`: A1区UPS设备DI类型第1个点位

## 设备管理

设备管理用于组织和管理机房设备。

### 查看设备列表
1. 点击左侧菜单"设备管理"
2. 查看所有设备及其状态
3. 可按设备类型、区域筛选

### 设备类型
- **TH**: 温湿度传感器
- **UPS**: 不间断电源
- **PDU**: 配电单元
- **AC**: 精密空调
- **DOOR**: 门禁系统
- **SMOKE**: 烟感探测器
- **IR**: 红外探测器
- **FAN**: 通风设备
- **LIGHT**: 照明控制
- **EPO**: 紧急断电

## 告警管理

告警管理帮助您及时发现和处理异常情况。

### 告警级别
| 级别 | 颜色 | 说明 | 响应时间 |
|------|------|------|----------|
| 紧急 (Critical) | 红色 | 严重影响系统运行 | 立即处理 |
| 重要 (Major) | 橙色 | 影响部分功能 | 30分钟内 |
| 次要 (Minor) | 蓝色 | 潜在问题 | 2小时内 |
| 提示 (Info) | 灰色 | 信息通知 | 可延后 |

### 查看告警
1. 点击左侧菜单"告警管理"
2. 默认显示活动告警
3. 可切换到"已确认"或"已解决"

### 处理告警

#### 确认告警
1. 选择要确认的告警
2. 点击"确认"按钮
3. 输入确认备注 (如：已查看现场)
4. 确认后告警状态变为"已确认"

#### 解决告警
1. 选择已确认的告警
2. 点击"解决"按钮
3. 输入处理结果 (如：已更换传感器)
4. 解决后告警状态变为"已解决"

### 告警统计
查看告警趋势和统计：
- 每日告警数量趋势
- 告警级别分布
- 高频告警点位排行

## 历史数据查询

历史数据查询用于分析趋势和生成报表。

### 查询历史数据

#### 1. 选择查询条件
- **点位**: 选择要查询的点位
- **时间范围**: 选择开始和结束时间
- **数据粒度**:
  - 原始数据: 按采集间隔
  - 分钟汇总: 1分钟平均值
  - 小时汇总: 1小时平均值
  - 日汇总: 1天平均值

#### 2. 查看结果
查询结果包含：
- **趋势图表**: 折线图或柱状图显示数据变化
- **统计信息**:
  - 最大值、最小值、平均值
  - 标准差、变化率
- **数据表格**: 详细数据列表

#### 3. 导出数据
支持导出格式：
- Excel (.xlsx)
- CSV (.csv)
- PDF (报表)
- JSON (原始数据)

### 数据对比
可同时选择多个点位进行对比分析：
1. 在点位选择中勾选多个点位
2. 系统自动生成对比图表
3. 可分析不同点位间的关联性

## 用电管理

用电管理模块帮助您监控能耗、优化用电，降低成本。

### 8.1 用电监控

#### PUE 监控
**PUE (Power Usage Effectiveness)** 是数据中心能效的关键指标。

- **计算公式**: PUE = 总功率 / IT负载功率
- **理想值**: 1.0 (不可能达到)
- **优秀值**: 1.2 - 1.5
- **一般值**: 1.5 - 2.0
- **需改进**: > 2.0

#### 功能特性
- **实时PUE仪表盘**: 大号仪表盘显示当前PUE
- **PUE趋势图**: 查看PUE历史变化
  - 小时趋势: 24小时
  - 日趋势: 30天
  - 周趋势: 12周
  - 月趋势: 12个月
- **功率分布**: 查看各设备的功率占比
  - IT负载功率
  - 制冷系统功率
  - UPS损耗
  - 照明功率
  - 其他功率

#### 设备实时功率
查看所有用电设备的实时数据：
- 电压 (V)
- 电流 (A)
- 功率 (kW)
- 功率因数
- 负载率 (%)

### 8.2 能耗统计

#### 日能耗统计
- **能耗汇总卡片**:
  - 总电量 (kWh)
  - 总电费 (元)
  - 平均功率 (kW)
  - 平均PUE
- **能耗趋势图**: 按小时显示能耗变化
- **峰谷平分布**: 饼图显示不同时段用电占比
- **电费分析**: 按峰谷平计算电费
- **同环比分析**: 与昨天、上周同期对比

#### 月能耗统计
- **能耗汇总**: 本月总电量、电费、平均PUE
- **每日趋势**: 柱状图显示每日能耗
- **同比环比**: 与上月、去年同期对比
- **数据表格**: 每日明细数据

#### 电价配置
可配置峰谷平电价：
- **峰时电价**: 默认 1.2 元/kWh (8:00-22:00)
- **谷时电价**: 默认 0.4 元/kWh (23:00-7:00)
- **平时电价**: 默认 0.8 元/kWh (其他时段)

配置步骤：
1. 进入"系统设置" → "电价配置"
2. 点击"新增电价"
3. 输入时段和价格
4. 保存配置

### 8.3 节能建议

系统会自动分析用电数据，生成节能建议。

#### 节能潜力卡片
- **潜在节能**: 预计可节省的电量 (kWh/年)
- **预计节省**: 预计可节省的费用 (元/年)
- **已完成建议**: 已采纳并完成的建议数
- **实际节能**: 已实现的节能效果

#### 建议类型
- **温度优化**: 空调温度设定建议
- **负载均衡**: 设备负载分配优化
- **时段调整**: 用电时段调整建议
- **设备更换**: 高能耗设备更换建议
- **能效提升**: PUE优化建议

#### 建议优先级
| 优先级 | 标识 | 说明 |
|--------|------|------|
| 高 | 红色标签 | 节能潜力大，建议优先实施 |
| 中 | 橙色标签 | 节能效果明显 |
| 低 | 蓝色标签 | 可延后实施 |

#### 处理建议

**接受建议**:
1. 查看建议详情
2. 评估可行性
3. 点击"接受"按钮
4. 制定实施计划

**拒绝建议**:
1. 点击"拒绝"按钮
2. 输入拒绝原因
3. 系统记录决策依据

**完成建议**:
1. 实施完成后点击"完成"
2. 输入实施说明
3. 系统计算实际节能效果

#### 节能效果跟踪
- 查看已完成建议的实际效果
- 对比预期节能和实际节能
- 生成节能报表

## 系统设置

系统设置用于配置系统参数和管理用户。

### 9.1 阈值配置

阈值用于触发告警判断。

#### 添加阈值
1. 进入"系统设置" → "阈值配置"
2. 点击"新增阈值"
3. 选择点位
4. 设置阈值参数：
   - **上限**: 超过此值触发告警
   - **下限**: 低于此值触发告警
   - **告警级别**: 紧急/重要/次要/提示
   - **延迟时间**: 持续多久后才触发 (防止误报)
5. 保存配置

#### 批量配置
可为同类型点位批量设置阈值：
1. 选择点位类型
2. 设置统一的阈值参数
3. 批量应用

#### 启用/禁用
- 可临时禁用某个阈值而不删除
- 禁用后该点位不会触发告警

### 9.2 用户管理

#### 用户角色
| 角色 | 权限 |
|------|------|
| 管理员 (Admin) | 所有功能，包括用户管理 |
| 操作员 (Operator) | 查看、操作、确认告警 |
| 观察员 (Viewer) | 仅查看数据 |

#### 添加用户
1. 进入"系统设置" → "用户管理"
2. 点击"新增用户"
3. 填写用户信息：
   - 用户名 (唯一)
   - 密码 (至少8位)
   - 姓名
   - 邮箱
   - 角色
4. 保存

#### 修改用户
- 可修改用户信息
- 可重置密码
- 可切换角色
- 可禁用/启用用户

### 9.3 系统日志

系统记录所有操作日志，用于审计和故障排查。

#### 日志类型
- **操作日志**: 用户操作记录 (登录、修改、删除等)
- **系统日志**: 系统事件记录 (启动、错误、警告等)
- **通讯日志**: API调用记录

#### 查询日志
1. 选择日志类型
2. 选择时间范围
3. 可按用户、操作类型筛选
4. 支持关键词搜索

#### 导出日志
- 可导出为 Excel 或 CSV
- 用于长期归档或外部分析

### 9.4 授权信息

查看系统授权情况：
- **许可证类型**: 基础版/标准版/企业版
- **点位限制**: 最多可监控的点位数
- **已用点位**: 当前使用的点位数
- **到期时间**: 许可证有效期

## 常见问题

### 10.1 无法登录

**问题**: 输入用户名密码后提示"用户名或密码错误"

**解决方案**:
1. 确认用户名和密码正确 (区分大小写)
2. 检查是否被禁用 (联系管理员)
3. 尝试重置密码
4. 清除浏览器缓存后重试

### 10.2 数据不更新

**问题**: 实时数据显示的时间很旧，不更新

**解决方案**:
1. 检查网络连接
2. 刷新浏览器页面 (F5)
3. 检查后端服务是否运行
4. 查看浏览器控制台是否有错误信息

### 10.3 告警没有声音

**问题**: 有告警但没有声音提醒

**解决方案**:
1. 检查浏览器是否允许播放声音
2. 检查系统音量设置
3. 首次访问时需要点击页面任意位置激活音频
4. 某些浏览器需要用户交互后才能播放声音

### 10.4 图表不显示

**问题**: 历史数据查询后图表不显示

**解决方案**:
1. 检查选择的时间范围内是否有数据
2. 尝试更换时间范围
3. 检查浏览器控制台是否有错误
4. 清除浏览器缓存

### 10.5 导出失败

**问题**: 点击导出按钮后没有下载文件

**解决方案**:
1. 检查浏览器是否阻止了下载
2. 检查数据量是否过大 (建议缩小时间范围)
3. 尝试更换导出格式
4. 检查磁盘空间是否充足

### 10.6 PUE 异常

**问题**: PUE 显示异常值 (如 0 或非常大)

**原因分析**:
- IT负载功率为 0 时 PUE 无法计算
- 数据模拟器可能生成异常数据
- 传感器故障

**解决方案**:
1. 检查功率传感器是否正常
2. 查看实时功率数据是否合理
3. 如果是模拟数据，重启数据模拟器

### 10.7 系统变慢

**问题**: 系统响应变慢，操作卡顿

**解决方案**:
1. 检查历史数据是否过多 (建议定期清理)
2. 关闭不需要的浏览器标签页
3. 检查系统资源占用 (CPU、内存)
4. 重启前后端服务

### 10.8 忘记密码

**问题**: 忘记登录密码

**解决方案**:
1. 联系管理员重置密码
2. 管理员可在"用户管理"中重置任何用户密码
3. 如果是管理员忘记密码，需要通过数据库直接修改
