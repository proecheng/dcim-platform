1. 系统概述

1.1 系统简介

机房动环监测系统（DCIM - Data Center Infrastructure Management）是一套专为数据中心设计的综合监控管理平台，实现对机房温湿度、电力、安防、设备等的实时监测、告警管理和数据可视化。

1.2 系统功能

实时监控

52个监控点位实时数据采集，支持AI/DI/AO/DO四种点位类型，WebSocket实时推送。

告警管理

四级告警机制（紧急/重要/次要/提示），声音提醒，实时弹窗通知。

用电管理

PUE实时监控，能耗统计分析，峰谷平电费计算，智能节能建议。

配电管理

配电系统拓扑配置，需量分析优化，负荷调节控制。

数据分析

历史数据查询，趋势分析，同环比对比，多格式报表导出。

系统管理

用户权限管理，阈值配置，操作日志，系统配置。

1.3 技术架构

层次 技术栈 说明
前端 Vue 3 + TypeScript + Element Plus 响应式界面，组件化开发
后端 Python FastAPI + SQLAlchemy 异步高性能API服务
数据库 SQLite / PostgreSQL 轻量部署或企业级部署
图表 ECharts 5 丰富的数据可视化
通信 WebSocket + REST API 实时推送 + 标准接口

1.4 版本历史

第二行显示五个交互式能源卡片，支持点击跳转和趋势展示：

3.2.5 节能建议卡片

  • 主要显示：待处理建议数量
  • 高优先级标识：紧急/高优先级建议数量

  • 点击操作：跳转到节能建议页面

  • 点位名称：人工可读的描述性名称
  • 类型：AI(模拟输入)/DI(开关输入)/AO(模拟输出)/DO(开关输出)

  • 点击"查看全部"跳转到告警管理页面

4. 点位管理

点位管理用于配置和管理系统中的所有监控点位。

4.1 点位类型

类型 全称 说明 示例
AI 模拟量输入 连续变化的测量值 温度、湿度、电压、电流
DI 开关量输入 二元状态值 设备运行状态、门禁状态
AO 模拟量输出 可调节的控制输出 温度设定、风速调节
DO 开关量输出 二元控制输出 设备启停、阀门开关

4.2 点位编码规则

点位编码格式：{区域}_{设备类型}_{点位类型}_{序号}

  • 区域：A1, A2, B1... 表示机房区域
  • 设备类型：TH(温湿度), AC(空调), UPS, PDU, ENV(环境), SEC(安防)
  • 点位类型：AI, DI, AO, DO
  • 序号：001-999 三位数字

示例：A1_TH_AI_001 表示 A1区域温湿度设备的第1个模拟输入点（温度）

4.3 点位操作

  • 新增点位：点击"新增"按钮，填写点位信息
  • 编辑点位：点击操作列的"编辑"按钮
  • 删除点位：点击操作列的"删除"按钮（需确认）
  • 批量导入：支持Excel文件批量导入点位配置
  • 批量导出：导出当前点位配置为Excel文件
提示：点位的启用/禁用状态会影响数据采集和告警检测，禁用的点位不会采集数据。

5. 告警管理

告警管理提供告警的查看、确认、处理和统计功能。

5.1 告警级别

级别 颜色 说明 响应要求
紧急 (Critical) 红色 严重故障，可能导致业务中断 立即响应，5分钟内处理
重要 (Major) 橙色 重要异常，需尽快处理 15分钟内响应
次要 (Minor) 蓝色 一般异常，需关注 1小时内处理
提示 (Info) 灰色 信息提示，供参考 可计划处理

5.2 告警状态

  • 活动 (Active)：告警正在发生，尚未处理
  • 已确认 (Acknowledged)：已知晓告警，正在处理中
  • 已解决 (Resolved)：告警原因已消除，问题已解决

5.3 告警处理流程

发现告警

系统检测到异常，生成告警记录，页面弹窗提醒并播放告警声音。

确认告警

操作员查看告警详情，点击"确认"按钮表示已知晓。可填写确认备注。

处理问题

根据告警信息排查和解决实际问题。

解决告警

问题解决后，点击"解决"按钮关闭告警。需填写解决措施说明。

5.4 告警筛选

支持多维度筛选告警记录：

  • 按告警级别筛选
  • 按告警状态筛选
  • 按点位类型筛选
  • 按时间范围筛选
  • 按关键词搜索

8. 用电监控

用电监控页面提供数据中心电力系统的实时监控视图。

8.1 核心指标卡片

总功率

数据中心实时总功率消耗，单位kW。

IT负载

IT设备（服务器、网络、存储）功率。

制冷功率

空调、冷却系统的功率消耗。

PUE值

电源使用效率 = 总功率 / IT负载功率。

今日用电

当日累计用电量，单位kWh。

今日电费

当日累计电费，单位元。

8.2 PUE监控

PUE（Power Usage Effectiveness）是衡量数据中心能源效率的关键指标：

PUE = 数据中心总功率 / IT设备功率
PUE范围 评级 说明
≤ 1.4 优秀 能效领先，运营成本低
1.4 - 1.6 良好 能效较好，有优化空间
1.6 - 1.8 一般 能效一般，建议优化
> 1.8 警告 能效较差，需改进

8.3 PUE趋势图

显示PUE值随时间的变化趋势，支持多种时间粒度：

  • 小时：查看当日每小时PUE变化
  • 日：查看近期每日PUE趋势
  • 周：查看周度PUE平均值
  • 月：查看月度PUE统计

8.4 设备功率表格

显示各用电设备的实时电力参数：

参数 单位 说明
电压 V 设备输入电压
电流 A 设备负载电流
功率 kW 有功功率消耗
功率因数 - 有功功率与视在功率的比值
负载率 % 当前负载占额定容量的百分比

时段 时间范围 电价（参考） 优化建议

  • 同比：与去年同期对比（如本月 vs 去年同月）

正值表示增长↑，负值表示下降↓。

13. 需量分析

需量分析页面提供电力需量的监控和优化分析功能。

13.1 什么是需量

电力需量是指用户在一定时间间隔内的平均功率，通常以15分钟为计算周期。供电局按照申报需量或实际最大需量收取基本电费。

需量管理的意义：合理控制需量可以降低基本电费，避免超需量罚款。

13.2 15分钟需量曲线 V2.3 新增

显示近期的15分钟滑动窗口需量数据：

  • 蓝色曲线：实际需量变化
  • 红色虚线：申报需量限值
  • 橙色区域：超需量预警区间

13.3 需量配置分析

分析各计量点的需量配置合理性：

指标 说明
申报需量 向供电局申报的需量值
12月最大需量 近12个月的最大实际需量
平均需量 近期的平均需量值
利用率 实际需量占申报需量的百分比
建议需量 系统计算的建议申报值

13.4 需量优化方案

系统自动生成三种优化方案：

  • 保守方案：以历史最大值为基准，保留较大安全余量
  • 均衡方案：综合考虑成本和风险的推荐方案
  • 激进方案：以平均值为基准，最大程度降低基本电费

13.5 负荷转移分析

分析可转移负荷以削峰填谷：

  • 识别可转移的设备负荷
  • 计算转移后的需量变化
  • 评估转移的潜在节省
  • 生成负荷转移建议

14. 负荷调节 V2.3 新增

负荷调节页面提供可调设备的功率控制功能。

14.1 调节类型

类型 适用设备 调节方式
温度调节 精密空调、新风系统 调整设定温度影响压缩机功率
亮度调节 照明系统 调整照明亮度降低功耗
运行模式 UPS、空调 切换节能模式/标准模式
负载控制 非关键负载 调整设备负载率

14.2 调节配置列表

显示所有可调节设备的配置：

  • 设备名称：可调设备标识
  • 调节类型：温度/亮度/模式/负载
  • 当前值：当前设定或运行值
  • 调节范围：允许的调节区间
  • 当前功率：设备当前功率
  • 启用状态：是否允许调节

14.3 调节模拟

在实际调节前，可先进行模拟预测：

选择设备

选择要调节的设备。

设置目标值

通过滑块或输入框设置目标调节值。

查看模拟结果

系统预测调节后的功率变化和影响。

确认应用

确认无误后点击"应用"执行调节。

14.4 调节建议

系统基于当前运行状态自动生成调节建议：

  • 高峰时段削峰建议
  • 设备节能运行建议
  • 环境参数优化建议

每条建议显示预期节能效果和舒适度影响评估。

14.5 调节历史

记录所有调节操作的历史：

  • 调节时间
  • 操作人员
  • 调节设备和参数
  • 调节前后功率变化
  • 实际节能效果

16. 常见问题

Q1: 登录失败怎么办？

A: 请检查以下事项：

  • 确认用户名和密码是否正确（注意大小写）
  • 检查账号是否被禁用
  • 确认后端服务是否正常运行
  • 清除浏览器缓存后重试

Q2: 数据不更新怎么办？

A: 可能原因和解决方法：

  • 检查网络连接是否正常
  • 检查数据模拟器是否运行
  • 尝试手动点击"刷新"按钮
  • 检查浏览器控制台是否有错误

Q3: 告警声音如何关闭？

A: 在页面右上角点击声音图标可临时静音。如需永久关闭，在系统设置中配置告警通知方式。

Q4: 如何导出报表？

A: 在报表分析页面，选择报表类型和日期后点击"生成"，生成完成后点击"下载"按钮选择导出格式（PDF/Excel）。

Q5: PUE值异常怎么排查？

A: PUE异常可能原因：

  • IT负载功率采集点故障
  • 制冷系统功率异常增加
  • 其他辅助系统能耗增加
  • 检查配电计量点数据是否正常

Q6: 如何添加新的监控点位？

A: 进入"点位管理"页面，点击"新增"按钮，填写点位编码、名称、类型、单位等信息后保存。如需批量添加，可使用Excel导入功能。

Q7: 节能建议如何生成？

A: 系统会根据运行数据自动分析生成建议。也可在节能建议页面点击"智能分析"按钮手动触发分析。建议基于温度、负载、能耗等多维度数据综合评估。

Q8: 如何联系技术支持？

A: 如遇到无法解决的问题，请通过以下方式联系技术支持：

  • 查看项目文档：README.md
  • 提交Issue：GitHub项目页面
